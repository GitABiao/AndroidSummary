## 高级
因为水平有限，下面说说对高级工程师的一些能力要求。自然这些都是冰山一角，高级工程师的要求不止这些。
### 1、应用层高级工程师
      * Android系统全面理解
      * 各类开源框架的深入理解
      * 一些前端/后端知识
      * Linux简单理解
### 2、系统级高级工程师
      * 简单理解Linux内核，并能简单修改内核源码。
      * 深入理解Linux设备驱动，并能简单修改驱动代码。
      * 对c/c++有一定的编程能力，尤其智能指针这一块。
      * 对Android系统全面理解并能简单修改。
